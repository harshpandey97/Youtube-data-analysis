# 🌟 Project Overview

This project demonstrates a complete **YouTube Data Analytics pipeline**, beginning with cleaned datasets and ending with actionable business insights.

The project includes:

- 📥 Data Cleaning & Validation
- 📊 Exploratory Data Analysis (EDA)
- 💬 Sentiment Analysis on YouTube Comments
- 📈 Channel & Video Performance Analytics
- 📉 Business Insights
- 🤖 Machine Learning Ready Dataset
- 📋 Dashboard Visualizations

# 📈 Key Performance Indicators

| Metric | Description |
|---------|-------------|
| 📺 Total Videos | Number of analyzed videos |
| 👥 Subscribers | Aggregate subscriber count |
| 👀 Total Views | Total video views |
| 👍 Likes | Total likes |
| 💬 Comments | Total comments |
| 😊 Positive Sentiment | Percentage of positive comments |
| 📉 Negative Sentiment | Percentage of negative comments |
| 📈 Engagement Rate | Likes + Comments relative to Views |
