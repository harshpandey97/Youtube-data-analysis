# 🌟 Project Overview

This project analyzes YouTube channels, videos and comments to generate actionable insights. It includes data cleaning, exploratory data analysis, sentiment analysis and business intelligence dashboards.

# 🎯 Problem Statement

Creators and analysts need an easy way to understand channel growth, audience engagement and comment sentiment from YouTube datasets.

# 🎯 Objectives

- Clean raw datasets
- Analyze channel performance
- Study engagement metrics
- Perform NLP-based sentiment analysis
- Build dashboards
- Prepare ML-ready data

# ✨ Features

- Data Cleaning
- Exploratory Data Analysis
- KPI Dashboard
- Channel Analytics
- Video Analytics
- Sentiment Analysis
- Correlation Analysis
- Business Insights
- Machine Learning Ready Dataset
